# Quiz Scraper Studio

A premium web interface for scraping, cleaning, and editing quiz papers from `quizpractice.space`.

## Features

- **One-Click Scraping**: Paste a URL and get formatted data instantly.
- **Premium UI**: Built with Tailwind CSS for a modern look.
- **Live JSON Editor**: Edit the scraped data structure directly in the browser.
- **Paper Preview**: Visual preview of the quiz as it would look to students.
- **Image Support**: Automatically downloads and bundles images.
- **Export**: Download the complete package (JSON + Images).

## Setup

1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:

   ```bash
   python app.py
   ```

3. Open browser at `http://localhost:5000`.
